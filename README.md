## 🚀 Overview  
This project performs Exploratory Data Analysis (EDA) on a telecom dataset to identify key factors driving customer churn. It uncovers patterns related to tenure, contract type, pricing, and service usage, and provides actionable insights to improve customer retention.

---

## 🎯 Objective  
- Identify high-risk customers  
- Understand churn drivers  
- Provide data-driven insights  

---

## 📊 Key Highlights  
- 📉 **26.5% churn rate** (~1 in 4 customers)  
- ⏳ New customers → **~50% churn**  
- 📄 Month-to-month → **~45% churn**  
- 🌐 Fiber users → **~42% churn**  
- 💰 High charges → **~40% churn**  

👉 Churn is mainly driven by **tenure, contract type, pricing, and service usage**
